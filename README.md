## Visão gerada pelo NotebookLM atráves de suas fontes
As fontes descrevem recursos essenciais para o aprendizado e desenvolvimento na linguagem ``Python``, abrangendo desde cursos acadêmicos até documentação técnica. Um dos textos detalha o curso **CS50** da **Universidade de Harvard**, que ensina lógica de programação, estruturas de dados e o uso de ferramentas como o **Visual Studio Code**.
<br><br>
Outro recurso destacado é o repositório no **GitHub** que contém a tradução oficial da documentação para o português brasileiro, facilitando o acesso de falantes da língua ao conhecimento técnico. Os materiais explicam conceitos fundamentais como variáveis, funções, condicionais e tipos de dados, além de introduzir temas avançados como programação orientada a objetos.
<br><br>
O objetivo central é capacitar programadores iniciantes e veteranos por meio de instruções práticas e materiais de referência padronizados.

## 🧠 Engenharia de Prompt e "Cicatrizes"
Durante a interação com o **NotebookLM**, utilizei as seguintes estratégias:

- **Prompt inicial**: *Ensine todas as nuances de Python para um iniciante. O objeto é transformar essa pessoa, que atualmente não tem conhecimento, em um profissional expert na linguagem de programação.*
- **Evitar alucinações**: na criação do caderno temático, optei por salientar no prompt que o especialista gerado deve ensinar à uma pessoa sem conhecimento prévio.

## 🚀 Miniguia de estudo

### 📖 Glossário de conceitos

- **Variáveis e Tipagem**: Como o Python lida com dados de forma dinâmica.
- **Estruturas de Controle**: O uso de ``if``, ``else`` e loops (``for``, ``while``) para ditar o fluxo do programa.
- **POO (Programação Orientada a Objetos)**: Organização de código através de classes e objetos (essencial para o nível expert).
